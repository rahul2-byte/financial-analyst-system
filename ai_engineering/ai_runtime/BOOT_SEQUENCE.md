# BOOT SEQUENCE

When starting a coding session, you must:

1. Load PROJECT_CONSTITUTION.md
2. Load SYSTEM_ARCHITECTURE.md
3. Load CODING_STANDARDS.md
4. Load AGENT_RULES.md
5. Load ORCHESTRATION_RULES.md
6. Load LLM_USAGE_POLICY.md
7. Load ACTIVE_TASK.md

You must confirm understanding of:

- System boundaries
- LLM limitations
- Agent separation
- Modular architecture
- Deterministic computation rule

Before writing any code, you must:

- Summarize current task
- Identify impacted modules
- Propose implementation plan
- Await approval if architectural changes are detected
